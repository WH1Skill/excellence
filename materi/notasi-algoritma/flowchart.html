Pengertian Flowchart

Apa Itu Flowchart?

Flowchart adalah diagram yang digunakan untuk menggambarkan alur logika atau proses dalam bentuk simbol-simbol grafis. Ini membantu dalam mendokumentasikan dan memahami langkah-langkah suatu proses.

Simbol-Simbol Flowchart

Simbol Nama Fungsi
üî∑ Terminator Menunjukkan awal atau akhir dari proses.
üü© Proses Menunjukkan langkah atau perhitungan dalam proses.
üî∫ Decision Digunakan untuk kondisi bercabang, seperti "Ya" atau "Tidak".
üü¶ Input/Output Digunakan untuk menerima input atau menampilkan output.
üîΩ Arus Menunjukkan aliran logika dari satu langkah ke langkah lain.

2. Flowchart Menentukan Bilangan Ganjil atau Genap

Pseudocode
    Mulai
        Input angka N
        Jika N mod 2 = 0 maka
            Cetak "Bilangan Genap"
        Jika tidak maka
            Cetak "Bilangan Ganjil"
    Selesai
Implementasi dalam C++
    #include <iostream>
    using namespace std;
    
    int main() {
        int N;
        cout << "Masukkan sebuah angka: ";
        cin >> N;
        
        if (N % 2 == 0)
            cout << "Bilangan Genap" << endl;
        else
            cout << "Bilangan Ganjil" << endl;
    
        return 0;
    }
