Variabel dan Tipe Data

Apa Itu Variabel?

Variabel adalah wadah untuk menyimpan data dalam program. Setiap variabel memiliki nama dan nilai yang dapat berubah.

Contoh dalam kehidupan sehari-hari:

  • Nama seseorang → disimpan dalam variabel nama
  • Usia seseorang → disimpan dalam variabel umur
  • Harga barang → disimpan dalam variabel harga

Contoh Variabel dalam C++

    #include <iostream>
    using namespace std;
    
    int main() {
        string nama = "Andi";   // Variabel string
        int umur = 20;          // Variabel integer
        double tinggi = 170.5;  // Variabel desimal
        cout << "Nama: " << nama << ", Umur: " << umur << " tahun" << endl;
        return 0;
    }

Tipe Data dalam Pemrograman

Setiap variabel memiliki tipe data yang menentukan jenis nilai yang dapat disimpan.

Tipe Data Deskripsi Contoh
int Bilangan bulat int umur = 20;
float Bilangan desimal (kurang presisi) float suhu = 36.5;
double Bilangan desimal lebih presisi double pi = 3.14159;
char Satu karakter char huruf = 'A';
string Teks atau kumpulan karakter string nama = "Budi";
bool Benar atau salah (true/false) bool isAlive = true;

Contoh Program dengan Berbagai Tipe Data

    #include <iostream>
    using namespace std;
    
    int main() {
        int umur = 25;
        double tinggi = 175.5;
        char inisial = 'A';
        string nama = "Aldi";
        bool isMahasiswa = true;
        
        cout << "Nama: " << nama << ", Umur: " << umur << " tahun" << endl;
        return 0;
    }
