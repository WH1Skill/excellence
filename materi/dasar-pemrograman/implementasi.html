Implementasi Konsep Algoritma dalam Pemrograman

1. Sekuensial

Pada algoritma sekuensial, perintah dieksekusi satu per satu secara berurutan. Contohnya, untuk menghitung luas persegi panjang:

  • Input panjang
  • Input lebar
  • Hitung luas = panjang Ã— lebar
  • Cetak hasil luas
Contoh Implementasi dalam C++:
    #include <iostream>
    using namespace std;
    
    int main() {
        double panjang, lebar, luas;
        cout << "Masukkan panjang: ";
        cin >> panjang;
        cout << "Masukkan lebar: ";
        cin >> lebar;
        luas = panjang * lebar;
        cout << "Luas persegi panjang adalah: " << luas << endl;
        return 0;
    }

2. Percabangan

Percabangan memungkinkan program untuk mengambil keputusan berdasarkan kondisi tertentu. Misalnya, menentukan apakah sebuah angka positif atau negatif.

Pseudocode Percabangan (If-Else):
    Mulai
        Input angka N
        Jika N > 0 maka
            Cetak "Bilangan Positif"
        Jika tidak
            Cetak "Bilangan Negatif"
    Selesai
Implementasi dalam C++:
    #include <iostream>
    using namespace std;
    
    int main() {
        int N;
        cout << "Masukkan angka: ";
        cin >> N;
        if (N > 0)
            cout << "Bilangan Positif" << endl;
        else
            cout << "Bilangan Negatif" << endl;
        return 0;
    }

3. Perulangan

Perulangan (looping) adalah proses menjalankan blok kode berulang kali hingga kondisi tertentu terpenuhi. Contohnya, mencetak angka 1 sampai 5.

Pseudocode Perulangan (For Loop):
    Mulai
        Untuk i dari 1 sampai 5
            Cetak i
    Selesai
Implementasi dalam C++:
    #include <iostream>
    using namespace std;
    
    int main() {
        for (int i = 1; i <= 5; i++) {
            cout << i << endl;
        }
        return 0;
    }
