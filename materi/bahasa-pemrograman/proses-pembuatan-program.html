Proses Pembuatan Program

1. Kompilasi

Proses di mana kode sumber diubah menjadi file executable sebelum dijalankan. Ini membuat program berjalan lebih cepat karena sudah diterjemahkan ke bahasa mesin sebelumnya.

Contoh Program C++:

    #include <iostream>
    using namespace std;
    
    int main() {
        cout << "Hello, World!" << endl;
        return 0;
    }

Setelah dikompilasi dengan g++ program.cpp -o program, hasilnya bisa dijalankan langsung.

2. Interpretasi

Kode program dijalankan langsung tanpa dikompilasi, tetapi dieksekusi baris per baris oleh interpreter.

Contoh Program Python:

    print("Hello, World!")

Kode ini bisa langsung dijalankan dengan perintah python program.py.

3. Kombinasi Kompilasi & Interpretasi

Kode dikompilasi ke bytecode lalu dieksekusi oleh interpreter, seperti yang terjadi pada bahasa Java dan C#.

Contoh Program Java:

    public class Main {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
