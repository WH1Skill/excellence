Operator dalam Pemrograman

1. Operator Aritmatika

Operator aritmatika digunakan untuk melakukan operasi matematika dasar seperti penjumlahan (+), pengurangan (-), perkalian (*), pembagian (/), dan sisa bagi (%). 

    int a = 10, b = 3;
    int penjumlahan = a + b;   // 13
    int pengurangan = a - b;   // 7
    int perkalian   = a * b;   // 30
    int pembagian   = a / b;   // 3
    int sisa        = a % b;   // 1

2. Operator Logika

Operator logika menggabungkan kondisi boolean. Contohnya, AND (&&), OR (||), dan NOT (!). 

    bool kondisi1 = true, kondisi2 = false;
    bool hasilAnd = kondisi1 && kondisi2; // false
    bool hasilOr  = kondisi1 || kondisi2; // true
    bool hasilNot = !kondisi1;           // false

3. Operator Pembanding (Relasional)

Operator pembanding digunakan untuk membandingkan dua nilai. Contoh: ==, !=, >, <, >=, dan <=. 

    int x = 5, y = 8;
    bool sama = (x == y);      // false
    bool tidakSama = (x != y);  // true
    bool lebihBesar = (x > y);  // false
    bool lebihKecil = (x < y);  // true

4. Operator Penugasan

Operator penugasan memberikan nilai ke variabel. Contoh: =, +=, -=, *=, /=, dan %=. 

    int a = 10;
    a += 5;  // a = a + 5, hasilnya 15
    a *= 2;  // a = a * 2, hasilnya 30

5. Operator Bitwise

Operator bitwise melakukan operasi pada bit biner. Contoh: &, |, ^, ~, <<, dan >>. 

    int a = 6;   // biner: 0110
    int b = 3;   // biner: 0011
    int hasilAnd = a & b; // 2  (0010)
    int hasilOr  = a | b; // 7  (0111)
    int hasilXor = a ^ b; // 5  (0101)

6. Operator Khusus

Operator khusus memiliki fungsi spesifik. Misalnya:
Pointer: & untuk mendapatkan alamat memori dan * untuk mendereference.
Operator Ternary: ?: sebagai cara singkat percabangan. 

    /* Contoh Pointer */
    int x = 10;
    int *ptr = &x; // Mendapatkan alamat memori x
    cout << "Nilai x: " << *ptr << endl;
    
    /* Contoh Operator Ternary */
    int a = 5, b = 8;
    int max = (a > b) ? a : b;
